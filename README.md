## ğŸ› Troubleshooting

### Model files not found
- The app automatically generates them on first run
- Make sure you have write permissions in the project directory

### Port 5000 already in use
- Change the port in `app.py` line 85: `app.run(debug=True, host='localhost', port=5001)`

### Dependencies not installing
- Upgrade pip: `pip install --upgrade pip`
- Try installing with Python 3.8+: `python3 -m pip install flask scikit-learn pandas numpy`

### Predictions not working
- Make sure the text is at least 10 characters long
- Try with longer text for better accuracy
- Clear your browser cache and refresh the page

## â“ FAQ

**Q: What types of fake news can it detect?**  
A: It's trained to identify patterns in misleading claims, sensational language, and unverified information.

**Q: How accurate is the model?**  
A: ~90% accuracy on synthetic data, but real-world accuracy depends on article type and training data quality.
